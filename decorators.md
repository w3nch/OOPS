| **Decorator**                                              | **Where Used**      | **Description / Behavior**                                                       | **Example**                                                                                                                                                                 |
| ---------------------------------------------------------- | ------------------- | -------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `@staticmethod`                                            | Class method        | Method doesn’t receive `self` or `cls`. Can be called on class or instance.      | `python\nclass A:\n    @staticmethod\n    def hello():\n        return "Hi"\nA.hello()\n`                                                                                   |
| `@classmethod`                                             | Class method        | Method receives `cls` (class) as first argument. Can access class variables.     | `python\nclass A:\n    count = 0\n    @classmethod\n    def increment(cls):\n        cls.count += 1\n`                                                                      |
| `@property`                                                | Instance method     | Turns a method into a read-only attribute.                                       | `python\nclass A:\n    def __init__(self, x):\n        self._x = x\n    @property\n    def x(self):\n        return self._x\n`                                              |
| `@<property>.setter`                                       | Property            | Defines a setter for a `@property` attribute.                                    | `python\n@x.setter\ndef x(self, value):\n    self._x = value\n`                                                                                                             |
| `@<property>.deleter`                                      | Property            | Defines a deleter for a `@property` attribute.                                   | `python\n@x.deleter\ndef x(self):\n    del self._x\n`                                                                                                                       |
| `@functools.lru_cache(maxsize=128)`                        | Function            | Caches results of function calls to optimize repeated calls.                     | `python\nfrom functools import lru_cache\n@lru_cache(maxsize=32)\ndef fib(n):\n    return fib(n-1)+fib(n-2) if n>1 else n\n`                                                |
| `@functools.wraps(func)`                                   | Function decorator  | Preserves original function metadata (name, docstring) when writing a decorator. | `python\nfrom functools import wraps\ndef decorator(f):\n    @wraps(f)\n    def wrapper(*args, **kwargs):\n        return f(*args, **kwargs)\n    return wrapper\n`         |
| `@abstractmethod`                                          | Abstract Base Class | Marks a method as abstract; subclass must implement it.                          | `python\nfrom abc import ABC, abstractmethod\nclass A(ABC):\n    @abstractmethod\n    def do_something(self):\n        pass\n`                                              |
| `@dataclass`                                               | Class               | Automatically generates `__init__`, `__repr__`, `__eq__` etc.                    | `python\nfrom dataclasses import dataclass\n@dataclass\nclass Point:\n    x: int\n    y: int\n`                                                                             |
| `@singledispatch`                                          | Function            | Enables function overloading by type of first argument.                          | `python\nfrom functools import singledispatch\n@singledispatch\ndef process(val):\n    print("Default", val)\n@process.register(int)\ndef _(val):\n    print("Int", val)\n` |
| `@contextmanager`                                          | Function            | Turns a generator function into a context manager (`with` statement).            | `python\nfrom contextlib import contextmanager\n@contextmanager\ndef open_file(name):\n    f = open(name)\n    yield f\n    f.close()\n`                                    |
| `@staticmethod` / `@classmethod` combined with `@property` | Advanced            | Rarely used, e.g., `@classmethod @property` for class-level properties.          | `python\nclass A:\n    _val = 0\n    @classmethod\n    @property\n    def val(cls):\n        return cls._val\n`                                                             |
